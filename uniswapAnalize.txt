Задания для анализа позиций на uniswap.
1. Позиция (задается NFT) отслеживается на странице https://app.uniswap.org/positions/v3/unichain/59044. Далее будут похожие NFT/позиции.
Нужно написать скрипт sh или программу на каком-нибудь более подходящем языке (если это дает существенные преимущества) для извлечения некоторых данных из этой страницы и проведения простых вычислений.
1.1 В одном из полей страницы есть информация о размере позиции: "Позиция ... XXX,XX $"/
Нужно извлечь в переменную positionUSD.
1.2 В другом поле есть информация примерно следующая: "0,02344 WBTC = 1 WETH
(2 395,87 $)"
Нужно значение (в данном случае 2 395,87 $) извлечь в переменную ethRate.
1.3 В начале скрипта задать константу ethInitial=38.1
1.4 Вывести для сравнения значения 
Текущая стоимость начального эфира (= ethInitial*ethRate) и Текущее значение позиции (= positionUSD)
1.5 Вывести для сравнения значения
Начальное вложение эфира (= ethInitial) и Текущее значение позиции в эфирах (= positionUSD/ethRate)
